# Automated a11y tests

## Installation

### Node

Use the node version mentioned in `.nvmrc`

### Install deps

Run `npm install` to install all dependencies.

## Tests

### CLI Test

To run the tests on the CLI, use `npm run test`.

### Watch mode

To run the tests in Watch mode, use `npm run test:watch`.
